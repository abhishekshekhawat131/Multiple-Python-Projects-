A hands-on collection of Python projects built while transitioning into Data Analytics and Data Science.
Each project reflects a core concept — logic building, OOP, functional programming, or data handling —
progressing from fundamentals to real-world analytical thinking.

> "Every project here is a step forward in the journey from curiosity to capability."

## 🎮 Who Wants to Be a Millionaire? — Python Quiz Game

A terminal-based quiz game inspired by the iconic game show.
Built to strengthen Python fundamentals — loops, conditionals, lists, and input handling.
